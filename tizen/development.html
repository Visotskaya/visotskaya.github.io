    Разработка

    Tizen является программной платформой, основанной на открытом исходном коде, поддерживаемой ведущими операторами мобильной связи, производителями устройств и поставщиками микроэлектроники. Платформа Tizen предоставляет гибкую среду для разработки  приложений, основанных на технологиях HTML5, и нативных приложений.

    Tizen работает на множестве классов устройств. На данный момент вы можете разработывать для двух классов устройтсв: телефоны (Tizen SDK) и умные часы (Tizen SDK for Wearable).
    Для этого скачайте SDK для вашей операционной системе. В SDK входит IDE Eclipse, эмулятор и множество инструментов облегчающих вам разработку. Скачать
